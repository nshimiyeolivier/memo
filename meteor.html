INSTALLATION

    In a terminal

  1. curl https://install.meteor.com/ | sh
  2. $ meteor create ~/name of your proect
  3. $ cd ~/name of your proect
  4. $ meteor (pour lance le serveur)

    5. To run your new app:
    cd /home/user/my_cool_app
    meteor

    If you are new to Meteor, try some of the learning resources here

  5. Install react => meteor npm install --save react react-dom
  6. App running at: http://localhost:3000
